Build the Weirdest Auth System Ever

Forget boring logins. This YSWS is all about creativity and fun. Make an authentication system that's never been seen before. The wilder, the better!

Join Slack → Get Inspired

The Challenge

Build an esoteric, unique, or meme-worthy authentication system.
Or, create an auth system with a vulnerability and show how to exploit it.
No boring logins allowed!

  • • Make it wild, creative, or just plain weird
  • • Bonus: Add a fun twist, meme, or challenge
  • • Document your process and what makes it unique
  • • (Optional) Add a vulnerability and explain how to break it

Rules

  • No using libraries or services that do authentication from scratch for you (like Auth0, Firebase Auth, Supabase Auth, Clerk, etc).
  • You can use Firebase, Supabase, or similar platforms just for storing data (like user info, sessions, etc), but not for handling authentication logic.
  • Projects that implement local authentication (like storing users in a JSON or SQL database) will be highly rated!
  • Solutions must work for multiple users (not just a one-off demo for a single person).
  • No browser-only demos, your project should work for real users with real accounts.
  • It's totally fine to use basic session/cookie helpers (like Flask sessions, Express sessions, etc).
  • All code must be your own, and you should explain your creative process!
  • App must have a functionality! (not just authentication)

Example Ideas

Need inspo? Try one of these or invent your own!

  • • Log in by solving a riddle
  • • Authenticate with dance moves and pose detection
  • • Log in using your Minecraft server stats
  • • One-time password that's actually a rickroll
  • • Log in by clicking the right cat gif
  • • Auth via AI that decides if you're you based on your writing tone
  • • Voice authentication: sing a song to log in
  • • Take a photo of a random object (like a fork!)
  • • Build in a vulnerability and show how to hack it
  • • Meme-based password system
  • • Anything else that's never been done before!
